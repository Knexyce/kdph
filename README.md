KDPH (Knexyce Data Package Handler) is a Python tool for creating, encrypting, uploading, downloading, and extracting encrypted packages ("Knexyce Packages" or KPs). It uses Argon2id for key derivation and AES-GCM for authenticated encryption, supports large-file operations, and integrates with GitHub to publish and retrieve packages.
